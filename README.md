<h1>Identity &amp; Access Management (IAM) Implementation</h1>
<h3>AWS Enterprise Environment</h3>

<h2>Overview</h2>
<p>
  This repository documents the implementation of a secure and scalable IAM framework for a cloud-based organization operating in a regulated environment.
</p>
<p>
  The solution focused on enforcing least privilege, strengthening authentication, automating credential management, and enabling continuous security monitoring.
</p>

<hr/>

<h2>Key Responsibilities</h2>
<ul>
  <li>Designed role-based access control (RBAC) using IAM users, groups, and roles</li>
  <li>Enforced multi-factor authentication (MFA) for all human users</li>
  <li>Implemented automated user onboarding and group migration using AWS Lambda and Amazon EventBridge</li>
  <li>Built automated IAM access key rotation workflows</li>
  <li>Configured privileged access using AWS STS role assumption</li>
  <li>Enabled centralized logging and audit trails using AWS CloudTrail and Amazon CloudWatch</li>
  <li>Implemented real-time security alerting with Amazon SNS notifications</li>
  <li>Integrated Amazon GuardDuty and AWS Config for threat detection and compliance monitoring</li>
</ul>
